# JS Drum Kit

A simple browser-based drum kit that lets you play drum sounds using your keyboard.

## How It Works

Press the keys **A S D F G H J K L** to trigger drum sounds. Each key lights up with a visual effect when pressed.

| Key | Sound   |
|-----|---------|
| A   | Clap    |
| S   | Hi-hat  |
| D   | Kick    |
| F   | Open Hat|
| G   | Boom    |
| H   | Ride    |
| J   | Snare   |
| K   | Tom     |
| L   | Tink    |

## Getting Started

1. Clone or download this repository.
2. Open `index-FINISHED.html` in your browser.
3. Press the keys listed above to play sounds.

## Files

- `index-FINISHED.html` – The completed drum kit app.
- `index-START.html` – A starter file for building the app yourself.
- `style.css` – Styles for the drum kit UI.
- `sounds/` – WAV audio files for each drum sound.

## Technologies Used

- HTML
- CSS
- Vanilla JavaScript
